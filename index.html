Welcome to Full-Stack App

A full-stack demo with:

  • MySQL-persisted users
  • JWT authentication with server-side token blacklist on logout
  • Password reset flow (email → token → new password)
  • Role-based UI (Admin / User)
  • CRUD for Employees, Departments, Requests
⚠️ Note: Local CRUD data (employees, departments, requests) is stored in localStorage. User accounts are persisted in MySQL when the backend is running.
